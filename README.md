# PAHDS Connect

**PAHDS Connect** is a full-stack web application developed to support NGO operations for  
**People’s Action for Human Development Society (PAHDS)**.  
The platform digitally manages volunteers, projects, and social initiatives, helping NGOs improve transparency, accessibility, and operational efficiency.

## 📌 Project Description

PAHDS Connect is designed to help NGOs manage their social initiatives digitally.  
The application enables volunteers to register online, submit their details, and engage with NGO-led programs related to education, digital literacy, women empowerment, health awareness, and rural development.

The frontend is built using **React** to provide a responsive and user-friendly experience, while the backend uses **Node.js and Express.js** to handle API requests securely.  
A **MySQL database** is used for structured and reliable data storage.  
Overall, the project aims to simplify NGO operations and enhance community engagement through technology-driven solutions.
